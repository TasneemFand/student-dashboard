# student-dashboard built with react, react router, typescript, material ui, vite

- install the project:                                                                                                                                                                                                     
   npm install
  
- run the project:                                                                                                                                                                                                       
   npm run dev

## Dependencies:
- Material UI
- react hook form
- tanstack react query
- react-i18next
- material-react-table
- react router
- zod
